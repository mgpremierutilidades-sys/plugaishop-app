# MAXXIMUS — Visão geral (PC-only por enquanto)

## O que é
Um modo de trabalho para manter o projeto evoluindo com estabilidade e pouco esforço humano.

## Como funciona
1) Você dá uma meta em 1 frase
2) Eu transformo em backlog + flags + métricas
3) O sistema roda checks e registra resultados
4) Você acompanha por status/relatórios

## Por que PC-only agora
Mais simples, mais barato e mais rápido para começar.

## Quando entra notebook
Quando o ciclo estiver estável, o notebook vira “executor” para acelerar tarefas pesadas.
