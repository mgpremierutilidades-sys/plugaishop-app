# MAXXIMUS — Coleta de contexto (PC-only / Windows)

## Objetivo
Você faz 1 comando e eu ganho visão do projeto para assumir 99% do trabalho.

---

## Passo único (no terminal do VS Code)
1) Abra o terminal (PowerShell) na raiz `C:\plugaishop-app`
2) Rode:

- `npm run context`

---

## O que você faz depois
Você me manda UMA destas coisas (a mais fácil):

### Opção A (mais simples)
Cole aqui a saída do terminal inteira (Ctrl+A, Ctrl+C, cola aqui).

### Opção B (se o script gerar um arquivo)
Me diga:
- qual arquivo foi criado
- e cole o conteúdo dele aqui

> Eu vou te pedir o mínimo e só o essencial.

---

## Se der erro
Você só cola as últimas 20 linhas do erro aqui.
Eu devolvo correção + patch completo.
