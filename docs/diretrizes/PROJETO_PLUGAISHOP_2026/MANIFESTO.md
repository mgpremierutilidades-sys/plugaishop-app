# Manifesto — Plugaishop 2026

## Norte
**Shopping Center Digital Vivo** (social commerce + descoberta + confiança + conversão).

## Princípios
1. **UX premium**: fricção mínima, previsibilidade máxima.
2. **Performance é feature**: lento = quebrado.
3. **Confiança**: clareza, consistência, segurança.
4. **Métricas mandam**: toda mudança mede impacto.
5. **Incremental**: nada de reescrever do zero.

## Regras de Produto
- Toda ideia vira: **backlog + feature-flag + métrica**.
- **Layout congelado**: não mexer em estrutura/visual; apenas melhorias invisíveis (perf/robustez).
- “Carrinho” tem visual intocável; “voltar” = back.

## North Star (produto)
- **Compras concluídas com satisfação** (proxy: `purchase_success` + retenção).

## Métricas base (mínimo)
- Conversão (view→purchase)
- Add-to-cart rate
- Checkout start rate
- Crash-free sessions
- Tempo de interação (TTI)
