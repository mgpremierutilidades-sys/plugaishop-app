# Interseção — Plugaishop ↔ Maxximus

## Plugaishop fornece ao Maxximus
- Objetivos e métricas (North Star + KPIs)
- Restrições rígidas (layout congelado, incremental)
- Eventos padrão (view/click/add_to_cart/checkout_start/purchase/success/fail)

## Maxximus fornece ao Plugaishop
- Execução contínua (cadência)
- Auditoria (logs, score, checklist)
- Rollback e prevenção de regressão

## Loop
1. Definir hipótese + métrica
2. Implementar sob flag
3. Medir
4. Promover ou reverter
5. Registrar decisão e aprendizado
